# Generate Lorem Ipsum

A modern, professional placeholder text generator built with React and TypeScript. This application provides high-quality, business-oriented lorem ipsum text to enhance your design and development projects.

## Features

- Generate up to 12 professional paragraphs of placeholder text
- Clean, business-focused content suitable for corporate and professional use
- Responsive design for seamless use across devices
- Fast and lightweight, built with Vite for optimal performance
- TypeScript for type safety and better development experience

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/HamedSadim1/generate-lorem-ipsum.git
   ```

2. Navigate to the project directory:

   ```bash
   cd generate-lorem-ipsum
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173` to view the application.

## Usage

1. Enter the number of paragraphs you want to generate (0-12).
2. Click the "Generate" button.
3. Copy the generated text for use in your projects.

## Technologies Used

- **React**: A JavaScript library for building user interfaces
- **TypeScript**: A typed superset of JavaScript for better code quality
- **Vite**: A fast build tool and development server
- **ESLint**: A tool for identifying and reporting on patterns in JavaScript/TypeScript code

## Scripts

- `npm run dev`: Starts the development server
- `npm run build`: Builds the project for production
- `npm run lint`: Runs ESLint to check for code issues
